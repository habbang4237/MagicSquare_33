## 3. 커버리지 진단 (DEF-002)

### 3.1 증상

```text
CoverageWarning: Module src/entity was never imported.
No data was collected.
```

### 3.2 원인

| 구분 | 값 |
|------|-----|
| 잘못된 `--cov` | `src/entity` (파일시스템 경로로 인식, import 트리와 불연결) |
| 실제 import | `entity.user` 등 (`pyproject.toml` `pythonpath = ["src"]`) |

### 3.3 권장 명령

```powershell
python -m pytest tests/entity/ --cov=entity --cov-report=term-missing
```

검증 결과: **13 passed**, entity **96%** line coverage.
